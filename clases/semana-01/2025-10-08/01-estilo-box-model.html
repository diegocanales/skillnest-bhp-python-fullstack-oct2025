Título Principal - H1

Propiedades aplicadas a H1:

  • color: white
  • background-color: black
  • text-align: center

Subtítulo Importante - H2

Propiedades aplicadas a H2:

  • color: red
  • text-decoration: underline
  • text-align: right

Subtítulo Secundario - H3

Propiedades aplicadas a H3:

  • color: blue
  • font-style: italic
  • text-align: center
  • background-color: lightyellow

Este ejercicio demuestra cómo aplicar diferentes propiedades CSS para estilizar títulos de manera efectiva. Cada nivel de título (h1, h2, h3) tiene su propio conjunto de estilos que los hace visualmente distintos y fácilmente identificables.

Conceptos clave aprendidos:

  • Diferencia entre color (texto) y background-color (fondo)
  • Uso de text-align para alinear contenido
  • Aplicación de text-decoration para efectos de texto
  • Uso de font-style para cursiva
