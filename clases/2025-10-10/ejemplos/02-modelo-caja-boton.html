📦 Modelo de Caja - Walkthrough Paso a Paso

🎯 ¿Qué es el Modelo de Caja?

Cada elemento HTML es una caja compuesta por 4 capas:

┌─────────── MARGIN (espacio externo) ──────────┐
│ ┌───────── BORDER (borde) ─────────────────┐  │
│ │ ┌─────── PADDING (espacio interno) ────┐ │  │
│ │ │                                      │ │  │
│ │ │         CONTENT (contenido)          │ │  │
│ │ │         width × height               │ │  │
│ │ │                                      │ │  │
│ │ └──────────────────────────────────────┘ │  │
│ └──────────────────────────────────────────┘  │
└───────────────────────────────────────────────┘
💡 Importante: El ancho total que ocupa un elemento es:
width + (padding × 2) + (border × 2) + (margin × 2)

💻 Walkthrough: Creando un Botón Paso a Paso

Paso 1: Contenido Base

button {
    width: 150px;
    height: 40px;
    background-color: #2196F3;
    color: white;
}

❌ Problema: El botón se ve muy pegado y simple

Paso 2: Agregar Padding (espacio interno)

button {
    width: 150px;
    height: 40px;
    padding: 10px 20px;  /* Arriba-abajo: 10px | Izq-der: 20px */
    background-color: #2196F3;
    color: white;
}

✅ Mejora: El texto ya no está pegado a los bordes

Paso 3: Agregar Border (borde redondeado)

button {
    width: 150px;
    height: 40px;
    padding: 10px 20px;
    border: 2px solid #1976D2;  /* Grosor | Tipo | Color */
    border-radius: 5px;         /* Esquinas redondeadas */
    background-color: #2196F3;
    color: white;
}

✅ Mejora: Bordes suaves y definidos

Paso 4: Agregar Margin (espacio externo)

button {
    width: 150px;
    height: 40px;
    padding: 10px 20px;
    border: 2px solid #1976D2;
    border-radius: 5px;
    margin: 10px;  /* Espacio alrededor del botón */
    background-color: #2196F3;
    color: white;
}

✅ Mejora: Los botones están separados entre sí

Paso 5: Botón Final con Hover

button {
    /* ... todas las propiedades anteriores ... */
    transition: all 0.3s ease;
}

button:hover {
    background-color: #1976D2;  /* Color más oscuro */
    transform: translateY(-2px); /* Se eleva */
    box-shadow: 0 4px 8px rgba(0,0,0,0.2);
}

✅ ¡Botón interactivo y profesional!

📐 Sintaxis de Padding y Margin

4 formas de escribir padding/margin:

1. Un valor: padding: 20px;

→ Todos los lados iguales (arriba, derecha, abajo, izquierda)

2. Dos valores: padding: 10px 20px;

→ Vertical | Horizontal (arriba-abajo: 10px | izq-der: 20px)

3. Tres valores: padding: 10px 20px 30px;

→ Arriba | Horizontal | Abajo

4. Cuatro valores: padding: 10px 20px 30px 40px;

→ Arriba | Derecha | Abajo | Izquierda (sentido de las agujas del reloj ⏰)

💡 Tips Importantes

  • Padding: Crea espacio DENTRO del elemento (afecta el fondo)
  • Margin: Crea espacio FUERA del elemento (separa de otros)
  • Border: Línea entre padding y margin
  • box-sizing: border-box; hace que width incluya padding y border
  • Usa DevTools (F12) para visualizar el box model de cualquier elemento
